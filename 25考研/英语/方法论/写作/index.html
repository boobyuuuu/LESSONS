    方法论¶

    1. 背诵：10大必背范文

      背文章比背段落好，背段落比背句子好，背句子比背词汇好。

      注：背得不熟，等于没背！

    2. 四大背诵方法：

      1. 搞清逻辑脉络

      2. 背熟中文译文！！

      3. 多听英文录音

      4. 进行中英互译

    3. 默写：仔细对比原文，改正写作错误。

      大作文15分钟默写完

      小作文8分钟默写完，没有错误

    4. 仿写

    5. 印象原则

      书写工整，卷面整洁，字迹清晰

      练字！！！

      写作用笔：0.7 或 1.0 黑色直液笔

      千万不要有任何修改

    6. 高分作文三大特点：

      没有模板

      没有错误/很少

      没有跑题

    7. 低分作文：

      模板太重

      卷面太差

      错误太多

      跑题太远
